## Commands to start coding with Prisma

### 1. Install @prisma/client

```sh
npm install @prisma/client
```

### 2. Start Prisma

```sh
npx prisma init
```

### 3. Create a new database schema

```sh
npx prisma migrate dev --name init
```

### 4. Generate Prisma Client

```sh
npx prisma generate
```
